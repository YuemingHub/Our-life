# 🎵 音乐放这里

把你的歌放到 **本文件夹(`music/`)** 下。

## 推荐文件名

```
music/song.mp3
```

如果你换文件名,记得去 `../data.js` 里修改 `music.src`。

## 选歌建议

1. **格式**:`.mp3` 最通用,`.m4a` / `.ogg` 也行
2. **大小**:尽量 **5MB 以内**,太长会加载慢
3. **时长**:3-5 分钟的循环最好(代码会自动循环播放)
4. **风格**:安静、温柔优先,否则会盖过打字机的氛围
5. **版权**:自己剪辑/翻唱的,或者已购版权的,避免纠纷

## 找不到歌?

可以把歌上传到百度网盘,拿到分享链接后,把 `data.js` 里的:
```js
src: "music/song.mp3"
```
改成:
```js
src: "https://你的网盘直链.mp3"
```

## 注意

- 现代浏览器为了用户体验,**默认禁止自动播放**音乐。
- 用户在门禁页点"打开"按钮时,会同时触发音乐播放(已经符合浏览器策略,不会被拦截)
- 如果还是没声音,点击页面**右下角的 ♪ 按钮**手动播放
